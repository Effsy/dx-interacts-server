# dx-interacts-server

## Setup Instructions

#### Setup DxInteracts

Clone the DxInteracts [repo](https://github.com/Effsy/dx-interacts)

Follow their [README](https://github.com/Effsy/dx-interacts/blob/master/README.md) to deploy the contracts locally.


#### Setup the server

Clone this repo

Build the server

`> ./gradlew build`

Run the server

`> ./gradlew run`
